    Hierarchy

    This page explains the structure of Radiogaga firmware. It will be explained how and where different modules are instantiates.

    On the figure 1 we can see where each module is instantiates. And on the figure 2 we have a more detailled view of different interconnection between 2 board.

    • RADIOGAGA_TOP.vhd is top file of the project.
    • processing_top.vhd instantiates all processing modules, dsd, rsr, processing etc.
    • processing.vhd instantiates processing and radiogaga.
    • radiogaga.vhd instantiates all signal processing modules for radiogaga wich are located in PROCESSING_MODULES directory.
